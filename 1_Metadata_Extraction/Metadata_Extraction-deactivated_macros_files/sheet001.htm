Calculations Applied onto Base Features:
A. Cumulative Total: The daily sum of all pre-existing occurances and new occurances of the current date.
B. Smoothing: Also known as 7-day rolling/moving average, smoothing is a statistical method used to analyze and represent a set of data over a period of time. It is the averaging of each data point along with its 6 previous data points to create a new average for each day. Smoothing can account for fluctuations in data over time by smoothing out any short-term anomalies, spikes, or outliers. It is a preventative measure for diminishing any skewing of the results.
C. Occurance Rate: The number of new occurances of a feature in a population over a specified period of time expressed as a rate per proportion size of 100, 1,000, 100,000, or 1,000,000. This metric allows for easier comparison between 2 different populations as long as the specified period of time and proportion sizes are identical for both populations, and hence, allowing for constants to exist as a common baseline in both populations. The formula would be as follows: (new_occurances / population) * proportion size. 
D. Cumulative Occurance Rate:  The number of pre-existing and new occurances of a feature in a population over a specified period of time expressed as a rate per proportion size of 100, 1,000, 100,000, or 1,000,000. The formula would be as follows: ((existing_occurances + new_occurances) / population) * proportion size. Note that in this dataset, the totaling of certain features have already been provided as separate features (cumulative totals), i.e., we are not required to sum the existing and new occurances together in these instances, e.g., total_deaths.
